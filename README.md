# Portfolio Web

## Descripción

Este proyecto consiste en el desarrollo de un sitio web de portafolio personal como parte del Sprint 4 del programa de Desarrollo Web de TripleTen.
El objetivo es crear una página responsiva que presente información profesional, proyectos destacados y enlaces de contacto.

El diseño se basa en un prototipo proporcionado en Figma y se implementa utilizando HTML y CSS con un enfoque mobile-first y buenas prácticas de maquetación.

## Tecnologías utilizadas

HTML5 semántico
CSS3
Flexbox
Media Queries (Responsive Design)
Metodología de organización de estilos por secciones
Fuentes locales (Open Sans y Archivo Black)
Normalize.css

## Diseño Responsivo

El proyecto está diseñado para adaptarse a los siguientes puntos de ruptura:

1440px (Escritorio)
768px (Tablet)
320px (Móvil)
